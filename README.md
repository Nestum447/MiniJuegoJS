# 🎮 Mini Juego Avanzado con Joystick Virtual

Este es un mini-juego hecho en **JavaScript + HTML Canvas**, donde controlas un cuadrado azul con un **joystick virtual** en móviles o arrastrando con el mouse en PC.  
El juego incluye **obstáculos, enemigos, sistema de puntos y niveles**, además de una animación simple del jugador.

## 🕹️ Cómo jugar

### En celular 📱
- Usa el **joystick azul** en la parte inferior izquierda:  
  - Arrastra el stick en cualquier dirección para mover el cuadrado azul.  
  - Suelta el stick para detener el movimiento.  

### En PC 💻
- Arrastra el joystick con el **mouse** para mover el cuadrado azul.  

---

## 🔴 Obstáculos
- Rectángulos rojos que bloquean al jugador.  
- No puedes atravesarlos; debes rodearlos para moverte libremente.

## 🟠 Enemigos
- Rectángulos naranjas que se mueven y rebotan en los bordes del canvas.  
- Si el jugador toca un enemigo, el puntaje y nivel se reinician.

## ⭐ Sistema de puntos y niveles
- El puntaje aumenta constantemente mientras te mueves.  
- Cada 20 puntos se sube un nivel.  
- Los niveles pueden servir para agregar dificultad en futuras mejoras (más enemigos, velocidad mayor, etc.).

## ✨ Animaciones
- El jugador tiene un **color azul pulsante** que cambia suavemente, dando sensación de animación.
